Estrutura básica do HTML

O HTML consiste em basicamente em duas partes: head e body.

Head é o cabeçalho, coisas que serão carregadas antes de exibir para o usuário, e body o corpo do texto, o que de fato vai aparecer para o usuário.

No head tem uma tag chamada title, que é o título da página.

Falando sobre Tags

i: altera a apresentação da fonte para itálico;

strong: altera a apresentação da fonte para negrito;

u: (underline) serve para sublinhar;

mark: utilizada para marcar um texto;

sup: usada para deixar o texto sobrescrito;

sub: define o texto subscrito. O texto subscrito aparece meio caractere abaixo da linha normal e às vezes é renderizado em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H 2 O;

small: define textos menores (como direitos autorais e outros comentários secundários);

hr: define uma quebra temática em uma página HTML, inserindo uma linha abaixo do texto (por exemplo, uma mudança de tópico);

del: define o texto que foi excluído de um documento. Os navegadores geralmente traçam uma linha no texto excluído;

abbr: define uma abreviatura ou um acrônimo, como "HTML", "CSS", "Mr.", "Dr.", "ASAP", "ATM". Dica1: Use o atributo global para mostrar a descrição da abreviatura/acrônimo ao passar o mouse sobre o elemento title. "The WHO was founded in 1948.";

input: especifica um campo de entrada onde o usuário pode inserir dados. Essa tag, diferentemente das outras, não tem fechamento, tipo um /input>;

Ex:

img: é usada para incorporar uma imagem em uma página HTML;

Atributos das Tags

São propriedades que uma tag pode ter. Existem os mais globais, os quais todas as tags vão ter, e outros específicos de determinadas tags.

Atributos mais genéricos:

id: identifica determinado elemento

style: serve para inserir comando CSS diretamente no elemento

Ex: style="color: blue;"

class: também voltado para o CSS, serve para padronizar tudo que estiver inserido no elemento.

Atributos mais específicos:

type: específico do input. Pode receber "text", "number", "color", entre vários outros valores possíveis.

src: específico da tag img. Ex: img src: "endereço da imagem". Voltar

Textos

h1 a h6: o h vem de header (cabeçalho) e é voltado para títulos e subtítulos.

p: para o texto em si, usa-se o "p" que vem de (paragraph - parágrafo).

blockquote: (bloco de citação) usado para citações, por exemplo, pois o navegador interpreta isso e aumenta o recuo.

Listas

Existem basicamente dois tipos de listas:

ol: (ordered list - lista ordenada) representa uma lista de itens ordenados. Os itens são indicados por números.

ul: (unordered list - lista sem ordem) representa uma lista de itens sem ordem rígida. Os itens são indicados por marcadores.

li: usado para representar um item que faz parte de uma lista.

Dentro de uma lista, você pode inserir outra lista.

POr meio do CSS é possível alterar os padrões das listas.

As listas são comumente utilizadas para fazer menu, mas para deixá-las horizontais é necessário o uso do CSS.

Links

A tag "a" (anchor - âncora) define um hiperlink, que é usado para vincular de uma página a outra.

O atributo mais importante do "a" é o href, que indica o destino do link.

Ex: Dio.me

Por padrão, o link vem azul e sublinhado. Isso pode ser alterado por meio do CSS.

Por padrão o link abre na mesma aba ("target=_self"), mas para mudar isso pode usar outro atributo, o "target=_blank".

Ainda dentro do "a", podemos usar o atributo title="texto qualquer". Assim, quando passar o mouse em cima do link, aparecerá um balãozinho com o texto digitado.

O link serve não apenas para sites externos, também pode ser utilizado para alternar páginas dentro do mesmo site. Nesse caso, no href, deve ser inserido o nome do arquivo da página desejada.

Para fazer o caminho de volta, preciso inserir um link no arquivo da página atual com o nome do arquivo da página anterior.

A boa prática recomenda que se forem páginas do mesmo site o ideal é que sejam abertas na mesma aba, a fim de manter o usuário nela.

Observações diversas

A margem entre títulos, subtítulos e parágrafos vem predefinida pelo navegador, mas pode ser ajustada por meio de CSS.

Posso inserir outras tags dentro do texto, tipo strong, para negritar determinada palavra.

A tag "font" foi usada no HTML 4 para especificar o tipo da fonte, o tamanho da fonte e a cor do texto. Não é compatível com HTML5.Para definir essas caracteríticas no HTML5, devemos utilizar o CSS.
